백준 문제 풀이 repository   
   
- 난이도별 중분류 (Bronze < Silver < Gold < Platinum < Diamond < Ruby)   
- 각 문제 번호별 소분류   
- 단계 및 날짜는 코드 코드 상단 주석   

   
주석 예시   
C, C++, java   
```
// Date         : 20XX-XX-XX
// Problem      : No. 2557 Hello World (https://www.acmicpc.net/problem/2557)
// Difficulty   : Bronze 5
// Step         : 1
// Language     : C99
// Source Link  : https://www.acmicpc.net/source/xxxxxxxx
```
